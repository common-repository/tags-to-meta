Increase SEO by putting blog tags and categories into the meta keywords on each page

== Description ==

Puts the category and tags of the post in the meta keywords tag in the html to increase SEO compatability.  If not tags are found, the full list of tags on the site are used.

== Installation ==

1. Upload `tagstometa.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= Do I need to make any changes to my template? =

No, the plugin automatically enters the code in the header of each page.
